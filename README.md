# Mehram - Video Generator

A powerful video generation application that creates videos from various input sources.

## Features

- 🎬 Generate videos from templates
- 🎨 Customizable styling and effects
- 🔄 Batch processing support
- ⚡ Fast and efficient video generation
- 📦 Easy-to-use API

## Installation

```bash
git clone https://github.com/rameshprasad2254/Mehram.git
cd Mehram
npm install
```

## Usage

```bash
npm start
```

## Development

```bash
npm run dev
npm run test
```
